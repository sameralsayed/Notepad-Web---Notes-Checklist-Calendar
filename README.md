# Notepad Web - Notes, Checklist & Calendar

📝 A beautiful, fast, and fully functional web notepad with checklists and calendar integration — inspired by popular mobile note apps.

![Notepad Web Preview](https://via.placeholder.com/800x400/4285f4/ffffff?text=Notepad+Web)

## ✨ Features
- 📄 **Unlimited Notes** — Create, edit, and auto-save text notes
- ✅ **Checklists** — To-do lists, shopping lists with checkboxes
- 📅 **Calendar Integration** — Simple calendar view with task addition
- ⭐ **Pin & Organize** — Easy access to important notes (via list)
- 💾 **Auto-Save** — Notes saved automatically in browser
- 📤 **Export Simulation** — Ready for PDF/share features
- 📱 **Fully Responsive** — Excellent on mobile, tablet, and desktop
- 🔒 **100% Private** — All data stored locally with localStorage

## 🚀 How to Use
1. Open `index.html` in any browser
2. Switch between Notes, Checklists, and Calendar tabs
3. Create new notes or checklists
4. Click on notes to edit
5. Add tasks directly from the calendar demo

## 🛠️ Built With
- HTML5
- CSS3
- Bootstrap 5
- jQuery
